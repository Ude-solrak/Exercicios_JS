já o escopo de bloco ele é limitado aquele bloco específico.

4.
O código abaixo exibe "3" 3 vezes porque o laço "for" tem apenas uma variável i para o código inteiro, 
ao invés de criar uma para cada repetição pelo fato de ser utilizado "var" no lugar do "let", 
para corrigir o erro é só trocar "var" por "let".

for (var i = 0; i < 3; i++) { 
   setTimeout(() => console.log(i), 100); 
}

correção:

for (let i = 0; i < 3; i++) { 
   setTimeout(() => console.log(i), 100); 
}
